Краткая сводка возможностей Wine Chat 2

1. Что умеет система
- Искать вина в локальной базе `wine_cards_wide` (по названию, региону, рейтингу, сортам, винтажу, цвету, сахарности и др.).
- Отвечать на вопросы о винах и делать списки/топы.
- Делать web-поиск по винной теме (цены/наличие/описания) и формировать ответ без ссылок.
- Работать с публичными пользовательскими записями по винам: лайки и заметки.

2. Публичные записи (доступны всем пользователям)
- Формат записи: `user`, `record_type` (`like` или `note`), `content`, `wine_id`.
- `wine_id` хранится внутри системы; пользователю не обязательно знать id или url.
- Пользователь может назвать себя сам. Это не обязательно.
- Если имя не передано:
  - при наличии внешнего id запись идет от `ext:<external_user_id>`;
  - иначе от `Гость`.
- Контекст записи:
  - можно указать вино по названию и производителю;
  - можно сослаться на позицию в последнем списке (`позиция 2`, `номер 3`, `1`);
  - можно выбрать несколько позиций сразу (`1 и 2`, `1,2`, `позиции 1 3 5`);
  - можно выбрать диапазон позиций (`с 3 по 5`, `от 3 до 5`, `3-5`, `с третьей по пятую`);
  - можно выбрать все позиции (`все позиции из списка`, `все из списка`, `все найденные`, `все`);
  - можно выбрать начало/конец списка (`первые 3 позиции`, `последние 2 позиции`);
  - если найдено несколько вин, система покажет кандидатов и попросит выбрать номер.

3. Шаблоны команд для лайка и заметки
- Поставить лайк:
  - `Поставь лайк вину Совиньон Блан Денисов Вайнери`
  - `Отметь лайком вино Cosaque Магнум`
  - `Поставь лайк позиции 2`
  - `Сделай отметку для вин 1 и 2 из списка`
  - `Сделай отметку для всех позиций из списка`
  - `Поставь лайк позициям с третьей по пятую`
- Добавить заметку:
  - `Добавь заметку для вина Совиньон Блан Денисов Вайнери: Хорошая минеральность`
  - `Заметка для вина Cosaque: Плотное, фруктовое, хорошо к стейку`
  - `Заметка для позиции 1: Хороший аперитив`
  - `Заметка для позиций с 3 по 5: Хорошее соотношение цена/качество`

4. Примеры других операций с публичными записями
- Показать заметки по вину:
  - `Покажи заметки по вину Cosaque`
- Показать лайки по вину:
  - `Сколько лайков у вина Новый Свет. Шардоне`
- Показать все записи по вину:
  - `Покажи все публичные записи по вину Совиньон Блан Денисов Вайнери`
- Фильтр по пользователю:
  - `Покажи заметки пользователя Алексей по вину Cosaque`

5. Дополнительные примеры по каталогу
- `Покажи российские вина с рейтингом выше 90 за 2026 год`
- `Найди красные сухие вина Кубани до 3000 рублей`
- `Какие вина из сорта Рислинг оценены в 2025 году`
- `Покажи полный список вин с рейтингом 92`

6. Соответствия русских/латинских написаний
- `шато ле гранд восток` <-> `Chateau le Grand Vostock`
- `абрау-дюрсо` <-> `Abrau-Durso`
- `эссе` <-> `Esse`
- При поиске производителя система должна учитывать оба варианта написания.

7. Как получить эту сводку
- Запросите: `Покажи возможности`
- Или: `Что ты умеешь?`
